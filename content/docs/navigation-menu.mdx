## Installation

<Steps>
  <Step>
    Install the Navigation Menu component with the nocta-ui CLI:

    ```bash lineNumbers title="Terminal"
    npx nocta-ui add navigation-menu
    ```
  </Step>
  <Step>
    Import the primitives you need for your navigation structure:

    ```tsx lineNumbers title="Import"
    import {
      NavigationMenu,
      NavigationMenuItem,
      NavigationMenuGroup,
      NavigationMenuLink,
    } from '@/components/ui/navigation-menu';
    ```
  </Step>
</Steps>
