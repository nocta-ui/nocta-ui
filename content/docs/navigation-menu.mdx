## API Reference

### NavigationMenu

Top-level menubar that coordinates trigger interactions and panel positioning across the navigation surface.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Items rendered inside the menubar. |
| `className` | `string` | `undefined` | Custom classes for the menubar. |
| `size` | `'md'` | `'md'` | Size variant for triggers. |
| `panelClassName` | `string` | `undefined` | Classes merged into the panel. |
| `panelWrapperClassName` | `string` | `undefined` | Classes for the panel wrapper. |
| `arrowClassName` | `string` | `undefined` | Classes for the arrow element. |
| `...props` | `Ariakit.MenubarProps` | - | Additional `Ariakit.Menubar` props. |

### NavigationMenuItem

Interactive trigger that can either navigate directly or reveal nested panel content.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `React.ReactNode` | - | Trigger label content |
| `children` | `React.ReactNode` | `undefined` | Panel content for the item |
| `href` | `string` | `undefined` | Link target when used as an anchor |
| `shift` | `number` | `0` | Horizontal panel offset |
| `placement` | `Ariakit.MenuStoreProps['placement']` | `'bottom'` | Preferred panel placement |
| `size` | `'md'` | `'md'` | Trigger size variant |
| `className` | `string` | `undefined` | Custom classes for the trigger |
| `render` | `Ariakit.MenuItemProps['render']` | `undefined` | Render override; auto-anchor for `href` |
| `...props` | `Ariakit.MenuItemProps` | - | Additional `Ariakit.MenuItem` props |

### NavigationMenuLink

Styled menu item optimized for secondary content inside the floating panel.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `React.ReactNode` | - | Link label text |
| `description` | `React.ReactNode` | `undefined` | Optional supporting text |
| `href` | `string` | `undefined` | Destination URL |
| `className` | `string` | `undefined` | Custom classes for the link |
| `render` | `Ariakit.MenuItemProps['render']` | `undefined` | Render override; auto-anchor for `href` |
| `...props` | `Ariakit.MenuItemProps` | - | Additional `Ariakit.MenuItem` props |

### NavigationMenuGroup

Collects related navigation links under a shared label within the floating panel.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `React.ReactNode` | `undefined` | Optional group heading |
| `children` | `React.ReactNode` | - | Group content or links |
| `className` | `string` | `undefined` | Custom classes for the group |
| `labelClassName` | `string` | `undefined` | Custom classes for the heading |
| `...props` | `Ariakit.MenuGroupProps` | - | Additional `Ariakit.MenuGroup` props |
