## Installation

<Steps>
<Step>
Install the Popover component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx nocta-ui add popover
```
</Step>
<Step>
Import the component:

```tsx lineNumbers title="Import"
import {
  Popover,
  PopoverTrigger,
  PopoverContent
} from '@/components/ui/popover';
```
</Step>
</Steps>
