## Installation
<Steps>
<Step>
Install the Alert component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx nocta-ui add accordion
```
</Step>
<Step>
Then import the components you need:

```tsx lineNumbers title="Import"
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent
} from './accordion';

```
</Step>
</Steps>
