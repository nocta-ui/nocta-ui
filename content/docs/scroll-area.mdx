## Installation

<Steps>
<Step>
Install the Scroll Area component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx @nocta-ui/cli add scroll-area
```
</Step>
<Step>
Import the pieces you need:

```tsx lineNumbers title="Import"
import { ScrollArea } from '@/components/ui/scroll-area';
```
</Step>
</Steps>

## Examples

### Default Scroll Area

<DocsTabs defaultValue="preview">
  <DocsTab value="preview" lineNumbers title="Preview">
    <ScrollAreaBasicDemo />
  </DocsTab>
  <DocsTab value="code" lineNumbers title="Code">
    ```tsx lineNumbers title="Scrollable Panel"
    const notifications = [
      { title: 'Product launch', time: '2m ago', body: 'Release goes live tomorrow at 9:00 AM.' },
      { title: 'New mention', time: '18m ago', body: 'Marta mentioned you in #design-system.' },
      // ...
    ];

    <ScrollArea className="h-64 w-full max-w-md">
      <div className="space-y-4">
        {notifications.map((notification) => (
          <div
            key={notification.title}
            className="rounded-lg border border-border/60 bg-card-muted/40 p-4 shadow-xs"
          >
            <div className="flex items-center justify-between text-sm font-medium text-foreground">
              <span>{notification.title}</span>
              <span className="text-foreground/60">{notification.time}</span>
            </div>
            <p className="mt-2 text-sm leading-relaxed text-foreground/70">
              {notification.body}
            </p>
          </div>
        ))}
      </div>
    </ScrollArea>
    ```
  </DocsTab>
</DocsTabs>
