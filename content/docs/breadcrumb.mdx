## Installation

<Steps>
  <Step>
    Install the Breadcrumb component using the nocta-ui CLI:

    ```bash lineNumbers title="Terminal"
    npx nocta-ui add breadcrumb
    ```
  </Step>
  <Step>
    Then import the components you need:

    ```tsx lineNumbers title="Import"
    import {
      Breadcrumb,
      BreadcrumbList,
      BreadcrumbItem,
      BreadcrumbLink,
      BreadcrumbPage,
      BreadcrumbSeparator,
      BreadcrumbEllipsis,
    } from '@/components/ui/breadcrumb';
    ```
  </Step>
</Steps>
