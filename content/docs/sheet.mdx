## Installation

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
Install the Sheet component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx nocta-ui add sheet
```
</Step>
<Step>
Import the component:

```tsx lineNumbers title="Import"
import {
  Sheet,
  SheetTrigger,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetDescription,
  SheetFooter,
  SheetClose
} from '@/components/ui/sheet';
```
</Step>
</Steps>
