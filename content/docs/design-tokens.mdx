## Colors

All colors are defined in the modern `oklch()` color space, giving precise control over contrast and perceptual brightness across light and dark themes.

<TokenTable category="color" tokens={colorTokens} />

## Shadows

Nocta UI defines a consistent shadow scale for elevation levels.
All shadows use subtle opacity to maintain depth without harsh contrast.

<TokenTable category="shadow" tokens={shadowTokens} />

Shadows are identical in both themes for consistent depth perception.

## Radius

Border radius tokens define how rounded corners appear throughout the system.

<TokenTable category="radius" tokens={radiusTokens} />

Modify this token to globally adjust the curvature of all components in your project.

## Motion

Component animations share a single easing curve to feel cohesive across the system.
Define it once in `app/global.css` and reuse it anywhere you need the default transition feel.

<TokenTable category="motion" tokens={motionTokens} />

Nocta UI hooks this token into animations, so updating the curve immediately affects every motion primitive.
