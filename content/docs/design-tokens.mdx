## Colors

All colors are defined in the modern `oklch()` color space, giving precise control over contrast and perceptual brightness across light and dark themes.

## Shadows

Nocta UI defines a consistent shadow scale for elevation levels.
All shadows use subtle opacity to maintain depth without harsh contrast.

| Token            | Description           | Value                                                                                                                                                                                                                                 |
| :--------------- | :-------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `--shadow-sm`    | Light elevation       | `0px 1px 1px -0.5px rgba(9, 9, 11, 0.025)`<br />`0px 3px 3px -1.5px rgba(9, 9, 11, 0.05)`<br />`0px 6px 6px -3px rgba(9, 9, 11, 0.05)`                                                                                                 |
| `--shadow-md`    | Medium elevation      | `0px 1px 1px -0.5px rgba(9, 9, 11, 0.025)`<br />`0px 3px 3px -1.5px rgba(9, 9, 11, 0.05)`<br />`0px 6px 6px -3px rgba(9, 9, 11, 0.05)`<br />`0px 12px 12px -6px rgba(9, 9, 11, 0.05)`<br />`0px 24px 24px -12px rgba(9, 9, 11, 0.05)`  |
| `--shadow-lg`    | Elevated card         | `0px 1px 1px -0.5px rgba(9, 9, 11, 0.025)`<br />`0px 3px 3px -1.5px rgba(9, 9, 11, 0.05)`<br />`0px 6px 6px -3px rgba(9, 9, 11, 0.05)`<br />`0px 16px 16px -8px rgba(9, 9, 11, 0.05)`<br />`0px 32px 32px -16px rgba(9, 9, 11, 0.05)`  |
| `--shadow-xl`    | Popover / modal level | `0px 1px 1px -0.5px rgba(9, 9, 11, 0.025)`<br />`0px 3px 3px -1.5px rgba(9, 9, 11, 0.05)`<br />`0px 6px 6px -3px rgba(9, 9, 11, 0.05)`<br />`0px 20px 20px -10px rgba(9, 9, 11, 0.05)`<br />`0px 40px 40px -20px rgba(9, 9, 11, 0.05)` |
| `--shadow-2xl`   | Highest elevation     | `0px 1px 1px -0.5px rgba(9, 9, 11, 0.025)`<br />`0px 3px 3px -1.5px rgba(9, 9, 11, 0.05)`<br />`0px 6px 6px -3px rgba(9, 9, 11, 0.05)`<br />`0px 24px 24px -12px rgba(9, 9, 11, 0.05)`<br />`0px 48px 48px -24px rgba(9, 9, 11, 0.05)` |

Shadows are identical in both themes for consistent depth perception.


## Radius

Border radius tokens define how rounded corners appear throughout the system.

| Token | Description | Value |
|:------|:-------------|:-------|
| `--radius-base` | Base border radius for all components | `0.175rem` |

Modify this token to globally adjust the curvature of all components in your project.
