## Installation
<Steps>
<Step>
Install the Kbd component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx @nocta-ui/cli add kbd
```
</Step>
<Step>
Then import the component:

```tsx lineNumbers title="Import"
import { Kbd } from '@/components/ui/kbd';
```
</Step>
</Steps>
