## Installation

<Steps>
<Step>
Install the Hovercard component with the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx @nocta-ui/cli add hovercard
```
</Step>
<Step>
Import the parts you need:

```tsx lineNumbers title="Import"
import {
  Hovercard,
  HovercardTrigger,
  HovercardContent
} from '@/components/ui/hovercard';
```
</Step>
</Steps>
