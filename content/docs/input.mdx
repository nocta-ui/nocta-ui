## Installation
<Steps>
  <Step>
    Install the Input component using the nocta-ui CLI:

    ```bash lineNumbers title="Terminal"
    npx @nocta-ui/cli add input
    ```
    </Step>
    <Step>

    Then import the component:

    ```tsx lineNumbers title="Import"
    import { Input } from '@/components/ui/input';
    ```
    </Step>
</Steps>
