
<Step>
Import the component:

```tsx lineNumbers title="Import"
import {
  AreaChart,
  AreaChartHeader,
  AreaChartTitle,

  AreaChartActions,
  AreaChartGraph,
  AreaChartLegend,
  AreaChartDataTable,
```
