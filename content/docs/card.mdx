## Installation

<Steps>
  <Step>
    Install the Card component using the nocta-ui CLI:

    ```bash lineNumbers title="Terminal"
    npx @nocta-ui/cli add card
    ```
  </Step>
  <Step>
    Then import the components you need:

    ```tsx lineNumbers title="Import"
    import {
      Card,
      CardHeader,
      CardTitle,
      CardDescription,
      CardContent,
      CardFooter,
      CardActions,
    } from '@/components/ui/card';
    ```
  </Step>
</Steps>
