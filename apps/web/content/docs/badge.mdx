## Installation
<Steps>
<Step>
Install the Badge component using the nocta-ui CLI:

```bash lineNumbers title="Terminal"
npx @nocta-ui/cli add badge
```
</Step>
<Step>
Then import the component:

```tsx lineNumbers title="Import"
import { Badge } from '@/components/ui/badge';
```
</Step>
</Steps>
